# Project Overview

This project was developed as part of the Artificial Intelligence Internship at CodeTech IT Solutions.

The Movie Recommendation Engine recommends movies based on the similarity of their story descriptions. It uses Natural Language Processing (NLP) techniques and Machine Learning to compare movie overviews and recommend similar movies.

The recommendation system follows a Content-Based Filtering approach using TF-IDF Vectorization and Cosine Similarity.

---

# Project Scope

The project includes:

- Loading a movie dataset
- Cleaning missing values
- Text preprocessing
- TF-IDF Vectorization
- Cosine Similarity Calculation
- Recommending similar movies
- Saving recommendation results
